Консульський облік — офіційна реєстрація громадянина України у посольстві або консульстві України за кордоном.

Тимчасовий

Для тих, хто живе за кордоном на навчанні, роботі чи має карту побиту.

Постійний

Для тих, хто офіційно виїхав з України на ПМЖ.

Для чого потрібен консульський облік?

  • Оформлення паспортів (дорослим і дітям)
  • Реєстрація народження, шлюбу, смерті
  • Оформлення довіреностей та нотаріальних заяв
  • Отримання довідок та витягів
  • Перебування на військовому обліку для чоловіків за кордоном
  • Можливість користуватися автомобілем без розмитнення протягом 60 діб

Чому це важливо?

Без обліку більшість консульських послуг недоступні.

Підтвердження легального перебування за кордоном.

Спрощує юридичні та сімейні питання без поїздок в Україну.
